# Maze Runner

A maze game made in Unity with C#. You control the player using keyboard keys and have to make your way through the maze while avoiding enemies.

## Features
- Player movement controlled via keys
- Pre-defined enemies that move around the maze
- Health system — enemies can hurt you
- Enemies use BFS (Breadth-First Search) pathfinding to chase/navigate toward the player

## Tech Used
- Unity
- C#

## How to Run
1. Clone or download this repo
2. Open the project folder in Unity
3. Open the main scene and hit Play

## Why I Built This
Wanted to practice game logic in Unity and get better at C#. The BFS pathfinding for enemies was the trickiest part — figuring out how to make them actually navigate the maze instead of just moving randomly.
